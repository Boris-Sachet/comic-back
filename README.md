## Environment variables
| Variable  | Type | Exemple value  | Description                      |
|-----------|------|----------------|----------------------------------|
| MONGO_USR | str  | `mogousr`      | Mongodb username                 |
| MONGO_PWD | str  | `mongopwd`     | Mongodb password                 |
| MONGO_URL | str  | `mongo:27017`  | Mongodb url                      |
